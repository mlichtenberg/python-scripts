These scripts demonstrate several different ways to find duplicate records within a dataset, using the fuzzywuzzy and recordlinkage toolkits.

These scripts have been tested with Anaconda distribution of Python 3.6.

https://docs.anaconda.com/anaconda/
https://www.anaconda.com/download/

They require the following Python libraries:

    pandas (included with the Anaconda distribution)
    fuzzywuzzy
    recordlinkage
